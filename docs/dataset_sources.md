This project integrates multiple medical imaging modalities and electronic health record (EHR) datasets to build an EHR-Imaging Documentation System. The datasets uploaded in this repository provide both structured clinical records and unstructured imaging data that can be combined for multimodal analysis.

## Imaging Datasets

1. Bone Scan For Head

   - Description: Binary classification dataset for detecting fractures in bone X-rays.
   - Source:https://www.kaggle.com/datasets/mariusmarin/bs-80k
   - Format: JPEG images.

2. Brain X-ray

   - Description: X-ray of the brain for tumor detection and segmentation tasks.
   - Source: https://www.kaggle.com/datasets/makohadharrenpius/brain-tumor-xrays-dataset
   - Format: JPEG images.

3. Brain CT Scans

   - Description: Brain CT images useful for Brain abnormality and Tumor detection.
   - Source: https://www.kaggle.com/datasets/murtozalikhon/brain-tumor-multimodal-image-ct-and-mri
   - Format: JPEG images.

4. Brain MRI

   - Description:  Brain MRI images useful for Brain abnormality and Tumor detection.
   - Sources:https://www.kaggle.com/datasets/murtozalikhon/brain-tumor-multimodal-image-ct-and-mri
   -  Format: JPEG images.


5. Normal Images
   - Description: Control group images without abnormalities.
   - Applications: Used for balancing datasets across modalities and providing baseline controls.

## EHR Datasets

1. The dataset used in this project is Synthea-style synthetic EHR data.
It follows the structure and conventions used in Synthea-generated datasets, but it was customized and refined for this project.
No real patient data was used.
